파이썬 및 html로 구현해보고 싶은 서비스


1.Mrs. Green Apple 의 소개 사이트를 내가 직접 만들어보자!
(1) 위키와 공식사이트 번역을 참고해 남들에게 어떤식으로 소개해보면 좋을까 고민해보기
(2) 창공때 구현하지 못했던 버튼 기능을 이용해 클릭하면 멤버마다 정보가 뜰수 있게 했으면 좋겠다.
(3) 전체적인 곡들을 한번 쭉 들어본 다음에 주관적인 나의 생각을 간략하게 작성해서 소개할수 있었으면 좋겠다.

1.1 구현해보고 싶은 모듈 목록

메인 화면
(1) Mrs. Green Apple의 대표 사진 및 웹 페이지 하단에 클릭 시 소개 페이지로 넘어가는 버튼 존재.
(2) 모듈 기능 구상
(2).1 메인화면에 화면 해상도 맞춰서 사진 삽입
(2).2 메인 화면 하단에 기능 별 버튼 삽입
(2).3 각각의 버튼은 클릭 시 독립적인 기능을 가진 웹사이트로 이동


버튼 별 기능 구상
1. 클릭 시, 멤버별 프로필 소개하는 사이트로 이동
2. 클릭 시, Mrs.Green Apple의 곡 분위기 및 앨범 목록을 연도 및 발매 순서로 링크, 곡의 느낌에 대해 주관적으로 느낀점을
적은 사이트로 이동
3. 클릭 시, Mrs.Green Apple의 공식 사이트와 팬카페로 각각 이동할 수 있는 버튼 기능
4. 클릭 시, 자유롭게 의견을 작성하고 쓸 수 있는 자유 게시판 기능의 사이트로 이동
